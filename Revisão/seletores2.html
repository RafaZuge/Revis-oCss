Seletor de Tipo (h1):
Aplica estilos a todos os elementos 
.

Seletor de Classe (.highlight):
Aplica estilos a todos os elementos com a classe highlight.

Seletor de ID (#unique):
Aplica estilos ao elemento com o ID unique.

Seletor de Atributo ([data-info]):
Aplica estilos a todos os elementos que possuem o atributo data-info.

Seletor de Atributo com Valor Específico ([type="button"]):
Aplica estilos a todos os elementos que possuem o atributo type com o valor button.

Pseudo-Classe :hover:
Aplica estilos aos elementos  quando são passados pelo mouse.

Pseudo-Classe :nth-child(odd):
Aplica estilos aos elementos 
  •  de índice ímpar.

Pseudo-Classe :first-child:
Aplica estilos ao primeiro 
     filho de seu pai.

Pseudo-Classe :last-child:
Aplica estilos ao último 
     filho de seu pai.

Pseudo-Elemento ::before e ::after:
Adiciona conteúdo antes e depois do conteúdo de elementos com a classe quote.

Seletor Combinador de Descendentes (.container p):
Aplica estilos a todos os 
     que são descendentes de .container.

Seletor Combinador de Filhos Diretos (.container > p):
Aplica estilos a todos os 
     que são filhos diretos de .container.

Seletor de Grupo (h2, .grouped):
Aplica estilos a todos os 
     e todos os elementos com a classe grouped.
