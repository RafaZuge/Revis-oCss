display: grid:
Define o contêiner como um grid, transformando seus filhos diretos em itens de grid.

grid-template-columns:
Define o número de colunas no grid e suas larguras. repeat(3, 1fr) cria três colunas de largura igual.

grid-template-rows:
Define o número de linhas no grid e suas alturas. 100px 200px 100px cria três linhas com alturas específicas.

grid-gap:
Define o espaço entre os itens do grid.

grid-template-areas:
Define áreas nomeadas no grid para layout mais intuitivo. As áreas são especificadas em um formato de string.

grid-area:
Associa um item do grid a uma área nomeada definida em grid-template-areas.

align-items:
Alinha os itens do grid no eixo transversal (vertical por padrão). No exemplo, o uso de display: flex e align-items: center faz isso para os itens internos.

justify-items:
Alinha os itens do grid no eixo principal (horizontal por padrão).

align-content:
Alinha as linhas do grid quando há espaço extra no eixo transversal.

justify-content:
Alinha as colunas do grid quando há espaço extra no eixo principal.
